## 目录

- `taskos/` — TaskOS 个人任务管理 Skill

## TaskOS Skill

基于「领域/项目/任务」三层 SOP 的通用个人任务管理 skill。

- 版本：v1.4.0
- 兼容：Claude Code / Cursor / OpenClaw / WorkBuddy 等支持 Anthropic 风格 skill 的 agent
- 数据格式：纯 Markdown + JSONL，跨 AI agent 可移植
